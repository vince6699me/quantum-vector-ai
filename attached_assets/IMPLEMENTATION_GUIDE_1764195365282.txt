# QuantumAI Services Popup and Pricing Implementation Guide

## Overview
This guide explains how to implement the services popup modal on the services page with detailed service information, and how to modify the pricing page to use Kenyan Shillings (Ksh).

## Part 1: Services Popup Implementation

### Current Implementation State
The services popup is already implemented in the ServicesSection component (`src/components/sections/ServicesSection.tsx`). Here's how it works:

### How the Services Popup Works

#### 1. Service Data Structure
Each service object in the `services` array contains comprehensive information:

```typescript
const services = [
  {
    id: "custom-llm",
    icon: Brain,
    title: "Custom LLM Development",
    description: "Build domain-specific language models...",
    category: "Advanced AI",
    detailedInfo: {
      overview: "...",
      features: [...],
      benefits: [...],
      useCases: [...],
      timeline: "8-16 weeks",
      pricing: "Starting from $50,000"  // Current USD pricing
    }
  }
];
```

#### 2. Modal State Management
The component uses React state to control modal visibility:

```typescript
const [selectedService, setSelectedService] = useState<typeof services[0] | null>(null);

const openModal = (service: typeof services[0]) => {
  setSelectedService(service);
};

const closeModal = () => {
  setSelectedService(null);
};
```

#### 3. Modal Rendering
The modal is conditionally rendered when `selectedService` is not null:

```typescript
{selectedService && (
  <div className="fixed inset-0 z-50 flex items-center justify-center p-4 bg-black/50 backdrop-blur-sm">
    {/* Modal content */}
  </div>
)}
```

#### 4. Modal Content Structure
The modal displays:
- Service icon and title in header
- Overview section
- Key features (with check icons)
- Benefits (with check icons)
- Use cases (in styled cards)
- Timeline and pricing cards
- Call-to-action buttons

#### 5. User Interactions
- Click "Learn More" button to open modal
- Click outside modal or close button to close
- Press Escape key to close modal
- Modal is fully responsive and accessible

### Converting Popup Pricing to Kenyan Shillings

To convert the pricing from USD to Ksh, update the `pricing` field in each service's `detailedInfo`:

**Current format:**
```typescript
pricing: "Starting from $50,000"
```

**Update to Ksh (approximate conversion rate: 1 USD ≈ 130 Ksh):**
```typescript
pricing: "Starting from Ksh 6,500,000"
```

Conversion estimates for all services:
- $50,000 → Ksh 6,500,000
- $25,000 → Ksh 3,250,000
- $35,000 → Ksh 4,550,000
- $30,000 → Ksh 3,900,000
- $20,000 → Ksh 2,600,000
- $15,000 → Ksh 1,950,000
- $40,000 → Ksh 5,200,000
- $45,000 → Ksh 5,850,000
- etc.
